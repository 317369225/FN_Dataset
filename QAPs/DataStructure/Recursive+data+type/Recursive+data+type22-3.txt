What are the high level programming languages?
Those are the programming languages that have a high abstraction away from the hardware. Unlike the assembly family of programming languages, the hardware description languages and machine code, every programming language listed below and others is a high level programming language: 1.C++ 2.Python 3.C# 4.Scheme 5.Go 6.Erlang 7.JavaScript 8.PHP 9.C 10.Pascal 11.Forth 12.Nearly every programming language you see in use today except in tightly resource constrained environments such as in embedded development where assembly is the order of the day
