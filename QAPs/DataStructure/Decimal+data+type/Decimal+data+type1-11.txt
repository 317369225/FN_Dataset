Javascript, it has become the dominant programming language of the web. All modern web browsers implement the Javascript language, thus making it the only one deployed on all personal computers and mobile devices around the world. Javascript has become a powerful, dynamic and fully-functional object-oriented programming language on par with more traditional languages like C++. Ruby, or Java. It has advanced features such as prototypal inheritance, modules, namespaces, anonymous (lambda) functions, and even metaprogramming.
