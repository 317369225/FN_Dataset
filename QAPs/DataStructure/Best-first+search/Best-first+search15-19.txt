How can one be well prepared to answer data structure/algorithm questions in interviews?
From a freshers point of view, you should be well versed with Arrays, Stacks, Queues, Linked Lists and Binary Trees. Arrays :- One-D, Two-D, Dynamic. Logic + Algorithm. Stacks :- How does recursiom implement stack. Algorithm. Queue :- Dequeue, Priority Queue using Heap. Linked List :- Single, Double, Circular. Logic + Algorithm. Binary Tree :- All the types along with basic definition. Along with AVL, Binary Search Tree, B Tree, B+ Tree etc. Algorithm. Along with that you should be well versed with Heaps. You should also know about the Search Algorithms and the Sorting Algorithms.... (more)
