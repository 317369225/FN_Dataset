How does one go about solving this area/integral problem?
Since the function changes at x=0  from f(x)≥g(x)  to g(x)≥f(x)  , you have to evaluate the function at two separate intervals: f(x)−g(x)  at [−1,0]  and g(x)−f(x)  at [0,2]  , where f(x)=x 3 +3x 2 +5x  and g(x)=x 3 +2x 2 +7x  . Evaluating the integrals: ∫ 0 −1 ((x 3 +3x 2 +5x)−(x 3 +2x 2 +7x))  and ∫ 2 0 ((x 3 +2x 2 +7x)−(x 3 +3x 2 +5x))  Reducing the equations, you get: ∫ 0 −1 (x 2 −2x)  and ∫ 2 0 (−x 2 +2x)=∫ 2 0 (2x−x 2 )  Solving the integrals: ∫ 0 −1 (x 2 −2x)=x 3 3 −x 2   [(0) 3 3 −(0) 2 ]−[(−1) 3 3 −(−1) 2 ]=43   and ∫ 2 0 (2x−x 2 )=x 2 −x 3 3   [(2) 2 −(2) 3 3 ]−[(0) 2 −(0) 3 3 ]=43   Combining the areas between the curves of both intervals, you have 43 +43 =83
