If FB apps are dying, do we build open graph-integrated external sites instead?
The most important thing is what do you have the app for. If you want to use it as a promotion and get new likes, it should still be in the tab of your page. You can use the fan gate easily to tell non-fans to like the page. As for the other apps, going outside Facebook makes sense, because apps and page tabs don't work on mobile phones (if you don't access the full Facebook via browser). So if you are interested in mobile users and don't have time/budget to build iOS and Android apps, building a mobile friendly website is a way to go
