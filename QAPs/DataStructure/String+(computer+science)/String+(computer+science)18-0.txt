Is "hello world" a string or strings?
"hello world" is one string. It doesn't matter what is inside the quotes. It could be one word, two words, or multiple words. "apple" is one string. "strawberries and oranges" is one string. You can tell by the single pair of quotes. What's tripping you up is that the contents of the string has multiple words. String is a computer science concept. There's only one of them. A string may (or may not) contain multiple words. I string could contain no words at all. "" "blahblahnotarealword" Those are both strings. "this is a string with multiple English words" That last one is still one string. All the examples in this answer are single strings.
