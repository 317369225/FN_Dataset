What is the next number in this sequence: 1, 11, 21, 1211, 111221, 312211, 13112221?
1113213211. This is a very interesting sequence. Though they are numbers, but the next number is not derived using any mathematics logic. The next number comes from reciting the last number in the following way: For example, let's say the number is 111221 as given below So while saying you say there are 3 1's, 2 2's and 1 1's. Therefore the next number becomes 312211 and this is the next number. Saying 312211, there is 1 3, one 1's, two 2's and two 1's. Thus the next number is 13112221. Following the same logic, the next number is 1113213211 .
