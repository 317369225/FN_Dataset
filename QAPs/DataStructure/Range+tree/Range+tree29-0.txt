How do you output the nodes from a binary search tree given a range?
Quite simple one, here is code. Logic is self explanatory, void PrintRange(Node *pRoot, int min, int max) {    if( !pRoot )        return;    if( pRoot->key >= min && pRoot->key <= max ) {        PrintRange(pRoot->lChild, min, max);        cout << pRoot->key << " ";        PrintRange(pRoot->rChild, min, max);    } else {        if( pRoot->key >= min )            PrintRange(pRoot->lChild, min, max);        if( pRoot->key <= max )            PrintRange(pRoot->rChild, min, max);    }} You can also imitate the implementation of inorder traversal class on input binary search tree. During initialization, point the cursor to a and move on till b.
