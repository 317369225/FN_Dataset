Does Huffman coding usually appear in competitive programming contest?
Yes, it does. Like it was already said in other answers - it's not the most common topic, but still you need Huffman codes quite often. Sometimes problem asks you about Huffman code in a straightforward way (that mostly happens in easy/educative contests), in some other cases you need to modify it a bit or to use general idea of "merging two smallest elements" for some similar problem. Talking about frequency - I've met ~10 related problems over last year.
