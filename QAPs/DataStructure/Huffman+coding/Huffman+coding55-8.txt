What do you think if a senior software developer can't write code?
A software developer by definition develops software. A senior software developer might be expected to develop more code per unit time then abnormal software developer, or maybe less if they have additional design, mentoring, and customer interfacing duties, but they are still expected to develop some code. Otherwise they would have another title “"product manager”, “manager”, maybe even architect (although I'm deeply suspicious of architects that don't develop any code).
