What do you think if a senior software developer can't write code?
The question isn’t clear. It’s not clear if the Senior Software Developer (SSD) can’t write code (at all) or isn’t good at writing code or can’t write code in one or more of the languages their team works in. It isn’t clear what the role of SSD entails in the organisation. Sometimes an SSD is a lead programmer and directly responsible for writing code while (possibly) supervising others in their coding as well as (probably) taking some overall responsibility for the structure and design of the application (or some part of a larger application).
