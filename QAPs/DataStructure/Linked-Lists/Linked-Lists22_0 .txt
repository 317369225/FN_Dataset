Can we delete the 1st node in the linked list?Certainly! There’s a couple of details here that are important to consider. Deleting the first node haphazardly can lead to a data structure that you can no longer reference. The most simple solution to this problem is to designed the “second” node of the linked list as the new “first” You can then delete the old first node and reclaim its memory if necessary. The website Learn-C has a pretty good description of how you would accomplish this, along with a good general description of how Linked Lists work! 88 Views
