Is it possible to have linked lists of different data types?In Java, you can do something like this List<Object> heterogenousLinedList; In other programming languages you can usually achieve this by creating a list of parent type objects. List<ParentClass> entries; This is semantically possible. Note however, you cannot do this for primitive types as there is no hierarchy defined.
