How can one find a pointer to the previous elements in a singly linked list without iterating from the beginning?If you are talking strictly about the singly linked list that keeps the information only of the elements that come next to it you cannot find previous pointer. But if you are concerned about the memory then you can attain it but using the xor list but the information stored in the pointer filed will be for both the node before and after it. So, with a memory of single pointer field you can use it as doubly linked list.
