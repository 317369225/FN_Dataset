What is the syntax for linked lists in Java in Eclipse?You can create a simple linked list in Java by using LinkedList class. You can then use methods like: add(Object obj) - appends an element to the end of the list add(int index, Object obj) - inserts an element at a specified index addFirst(Object obj) - adds an element to the head of the list addLast(Object obj) - adds an element to the end of the list Object getFirst() - returns the first element of the list Object getLast() - returns the last element of the list Object get(int index) - returns the element on the specified index clear() - removes all elements from the list remove(int index) - removes the element at the specified index removeFirst() & removeLast() - remove and return the first and last elements of the list respectively set(int index, Object obj) - replaces the element at the specified index with the obj. size() - returns the size of the list (number of elements) You can use it like this: List<String> linkedList = new LinkedList<String>(); linkedList.add("myName"); linkedList.add(1,"myName2"); linkedList.size(); linkedList.get(0); linkedList.removeLast(); You can also use Iterator interface or ListIterator to iterate over the list. 438 Views  View Upvotes
