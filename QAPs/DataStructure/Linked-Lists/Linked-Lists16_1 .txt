In a doubly linked list if a node is to be deleted between two nodes, how many links of the existing list have to be modified?Only 2 links in the existing list needs to be modified that are next pointer of the node which is before node to be deleted and the previous pointer of the node which is next to the node to deleted. We need to point next pointer of the node which is before the node to be deleted to the next node of the node to be deleted and one other is that we have to change the previous pointer of the next node of the node to be deleted to the previous node ot he node to be deleted.
