How do I understand doubly linked lists?Each node in the doubly linked list carries the address of both the next node and the previous node. The first node in the list contains “null” in the previous field where as the last node in the list contains “null” in the next field. Doubly linked list should be easy to understand if you have understood singly linked lists. If you want to learn more lists visit MIT OCW lectures on Data Structures. 30 Views
