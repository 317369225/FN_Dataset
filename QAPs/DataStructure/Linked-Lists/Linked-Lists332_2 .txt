What is disadvantage of using linked lists over arrays?Though not disadvantages trade off 1) Have to do Linear search in Link List, If array is sorted then it gives logn search 2) In systems which use processor cache or virtual memory, scanning an array is much faster if successive elements are stored in consecutive positions in memory, we can not get this benefits while using Link List Ref : Locality of reference 3) Memory Overhead is more in case of Link List 4) static array creation is Implicit unlike Link List. 5) dynamic array can be created(malloc) or destroyed(Freed) very easily unlike a typical Link List. 783 Views  View Upvotes
