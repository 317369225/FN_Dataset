What does node=node->next exactly do in a linked list?When you traverse an array (say arr), you write 'arr++' or 'arr=arr+1' because array is a contiguous allocation of memory. In the same way to traverse a linked list (which is not a contiguous memory) you need 'node=node->next' as the address of the next node is stored in the 'next' field of each structure .
