All of these answers are interesting! :) In addition to all of these, for me the most important health conscious application is workrave. What it does is, it wakes you up after the predefined time limit set by you by locking the screen and all I/O devices for some time duration. This way, you can go away from the computer, have a walk and come back. All of the options are configurable, and can be set by you. It also shows some exercises, so that you don't get Repetitive strain injury working for a long time. Workrave - For more info. To install : sudo apt-get install workrave This is useful for me, as i tend to work indefinitely on the computer screwing my back and eyes.
