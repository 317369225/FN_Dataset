Which data structure is best suited for making a simple phone book program that returns a number given users name and vice versa?
Using two simple sorted array of the name and phone number respectively will also do the job . Both the array can have the pointer to contact field . You can create a simple Struct with both the property in int. Adding , deleting or searching the phone number or name will take O(logN) complexity using Binary Search and you will have data size of N . It is also good DS if you want to show the list in pagination in some sorted order of name or phone number.
