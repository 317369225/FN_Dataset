When should I use a binary search tree over a hash map?Binary Search tree would take less memory space than Hashmap For up to a certain number of items, Binary search tree would provide faster lookup and insert than Hashmap. Hashmap spends a constant time in calculating hash, that binary Search tree doesn't have to.
