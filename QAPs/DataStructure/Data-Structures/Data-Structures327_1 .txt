What's the differences between database and data structure? When do we use each?A database is a system used to store information on secondary memory for an indefinite amount of time in such a way that retrieving that information is efficient and secure. A data structure is a logical construct that groups basic data types. A data structure can and is a component in a database system. Besides, since a database is a system, can offer a great range of functionalities. EDIT, more clarification: A database is a system and, from a programmer’s point of view, it is used as a service that is outside the application that is being developed.
