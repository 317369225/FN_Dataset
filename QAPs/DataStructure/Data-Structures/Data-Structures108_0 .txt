How do I insert multiple data in a single node of linked list using STL?The push_back() function is used to insert only single data. So, how can I add multiple data in a single node?[It is not exactly clear, what’s the structure of your linked list or you are using the predefined data structures of STL. I will answer in a generic sense. Let me know if you have any doubt/confusion] use push_back(make_pair(data_item1,data_item2)); //you can store a pair of data items //or push_back( { data_item1, data_item2.... } ); //you can store a a list of data items, not just a pair Make sure your list is defined is such a way that it can hold multiple datas- e.g. using ‘pair’ for example - inserting a pair of integers in a list (STL) #include <bits/stdc++.h> using namespace std; int main() { int n; cin>>n; list<pair<int,int> > l; for(int i=0;i<n;i++){ int x,y; cin>>x>>y; l.push_back({x,y}); //or l.push_back(make_pair(x,y)); } for(auto i :l) cout<<i.first<<" "<<i.second<<"\n"; return 0; } Ideone.com For inserting a list of integers (not just pair) , “{ , , , }” works pretty fine #include <bits/stdc++.h> using namespace std;   int main() {   list<list<int> > v;   v.push_back({1,2,3,4});   for(auto i:v){   for(auto j:i){ cout<<j<<" "; } cout<<"\n"; } return 0; } 80 Views · View Upvotes
