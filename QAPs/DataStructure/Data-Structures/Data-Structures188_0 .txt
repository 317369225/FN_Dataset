We can create our own music library, search songs , artists etc. Also download playlist and sync offline. Data is backbone of the Spotify app.Tool similar to Google Analytics is used to handle huge data. It acts as music recommendation system. Distributed large volumes of data is managed using HDFS. Spotify uses apache storm to for realtime computation and data processing. In 2014 Spotify chose Apache Crunch to handle overload of data. They implemented sharded join solution from the standard Apache Crunch library to solve MapReduce performance problems. here is whole blog post on the topic!
