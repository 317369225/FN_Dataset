What is the deep web and how do you access it?Originally Answered: What is the Deep Web?The deep web is a large section of the World Wide Web not indexed by regular search engines. We only access 10% of the internet via google, bing, yahoo etc whilst the other 90% is accessed through an onion router such as Tor (anonymity network) https://www.torproject.org/ which allows the user to access.onion web pages. These are typically used for sites within the deep web. Think of the deep web like an iceberg... The danger is, using Tor etc users IP addresses are masked and anonymity is thereby maintained - users can theoretically become involved in drug sales, pedophilia etc 97.9k Views  View Upvotes
