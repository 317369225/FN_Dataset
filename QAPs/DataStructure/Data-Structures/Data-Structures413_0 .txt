Does adding n elements to an ArrayList take O(n) time or O(n^2) time?According to Java Platform SE 8 documentation, adding n n elements to ArrayList requires O(n) O(n) . The add operation runs in amortized constant time, that is, adding n elements requires O(n) time. All of the other operations run in linear time (roughly speaking).
