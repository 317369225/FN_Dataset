How is a hash table different to a linked list or an array?
Arrays and linked lists are data structures that store a collection of elements in a specific order. In arrays this order is fixed and you access them sequentially by index number (like a number of numbered storage lockers in a row). In linked lists the elements can theoretically be in any place in memory, and each also has the address in memory of the next one. Think of a beaded bracelet. This means you cannot get to any element using an index number easily (=quickly), but you have to start at the first one each time to search for the one you want.
