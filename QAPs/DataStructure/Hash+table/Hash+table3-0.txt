What is a hash table?
For purists sake, lets get the definition of a Hash table out of our way first. Quoting Wikipedia, In computing , a hash table (also hash map) is a data structure used to implement an associative array, a structure that can map keys to values. A hash table uses a hash function to compute an index into an array of buckets or slots, from which the correct value can be found. Done? Now lets discuss what a Hash table is.
