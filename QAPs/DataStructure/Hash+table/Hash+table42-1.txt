Why doesn't C also provide default implementations for linked lists, trees, hash tables, sorting, etc. like Java does?
There are a lot of reasons why programming in Java can be easier than programming in C, and the extensive library that is part of the Java platform certainly does help. The reason that C doesn't have a similar library is that when C was coming of age, space (including RAM and disk storage) was at a huge premium, and something like the Java platform would have been considered far too bloated to drag along everywhere.
