What is the difference between HashSet, HashMap and hash table? How do they behave in a multi-threaded environment?
This is a common interview question. HashSet does not allow duplicate keys and is not synchronized, allows a null key (read HashSet (Java Platform SE 7 ) for more). HashMap is not synchronized but allows duplicate keys, it also allows one null key and as many null values as you like (read HashMap (Java Platform SE 7 ) for more). Hashtable is synchronized and allows duplicate keys, it also does not allow null keys or values (read Hashtable (Java Platform SE 7 ) for more).Toby Thain is correct about collections.
