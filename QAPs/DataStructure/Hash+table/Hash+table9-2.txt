How do I implement hash table and hash map in c++?
Short answer: Don't. Use the standard library's premade unordered_map class instead. There is NO possibility that you'll ever make one yourself which would work better / faster / more robust / etc. Just get hold of the boost 3rd party lib, which also includes this. Chapter 23. Boost.Unordered. Later versions of C++ (03 and later) include this in their standard libraries, so you need not get hold of a 3rd party lib.
