What do you mean by Hash Map and Hash Table?
An interface for HashMap may be found at calculus - Map. The hashing used in this class is a new form of hashing called Binary Hashing. Hash tables are implemented using balanced, binary trees. Note that HashMaps have been implemented as databases as well (see calculus - Map). For a discussion and example of a HashMap see calculus for Java - Guide - Maps. The difference between a Table and a Map is that a table has an embedded key whereas in a map the key is separate from the data. For a discussion of Tables see: calculus for Java - Guide - Tables.
