Why doesn't C also provide default implementations for linked lists, trees, hash tables, sorting, etc. like Java does?
C has taken the approach that it is a language and a basic library of OS and a few other services: C standard library.  It does not attempt to define any "standard" data structures beyond those that are a core part of the language, a few extensions for portability (stdint.h), or those needed to interact with the standard library (like FILE, size_t, etc). And you can write a completely "legal" - and useful - C program without using the standard library.
