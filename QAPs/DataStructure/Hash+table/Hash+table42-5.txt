Why doesn't C also provide default implementations for linked lists, trees, hash tables, sorting, etc. like Java does?
I think really, C, is everything you need, and nothing more. C is basically a portable assembler, that was the idea behind it, and as such it succeeded in a way that hardly any other technology has, C is so close to perfect, you'd think it was discovered, not invented. However, it's perfection is really only possible because it does so little, the more stuff that goes on top, the more decisions you make for the programmer, and remember those decisions, if a part of C, must be applied to everything from a microcontroller with 32k of RAM, all the way up to IBM mainframes and super computers...
