What is the best method for checking if a binary tree is a binary search tree?This solution is valid assuming that all the elements in BST are unique. Do an in-order traversal of the tree, and store every element in an array during the in-order traversal. Now check if this array is sorted. In-order(i, T) {  if(T!=null) {  In-order(i, left(T));  Array[i]=value(T);  In-order(i+1, right(T))  } } checkArray(Array) {  for(i=0;i<Array.size()-1;i++)  {  if(Array[i]>Array[i+1])  return false;  }  return true; } The time complexity for this will be O(n).
