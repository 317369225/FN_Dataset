What is the difference between a data type and a data structure?
A data type is a shorthand definition of how memory is used for storage. The simplest case is probably “byte”, where that type is defined. A “byte” is a small integer, 8 bits long, able to hold values from 0 to 255. A data structure is usually, in my experience, the term for what results when your program allocates memory. Usually it refers to an allocation that has more than one kind of data defined in it.
