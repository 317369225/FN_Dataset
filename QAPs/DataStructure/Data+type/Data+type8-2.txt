What's the difference between a data structure and an abstract data type?
ADT is a representation of a data type in terms of the data types possible values, operations permissible without concerning about it implmentation details. The term abstract here means that the description is generally in terms of the user of the data type. However Data Structures are concrete implementations and are more inclined toweds the point of view of the implementer. ADT is more of a theoretical concept with not all features supported by a programming language.
