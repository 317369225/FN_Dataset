What is the origin of data types in programming languages?
Type theory started, as much of computer science did, as an aspect of the foundations of mathematics. Russell's paradox arises from the idea that if you have some mathematically well-defined property P, then you can have the set "the set of all x such that P(x)." Bertrand Russell thought, "Consider then the set of all sets that don't contain themselves. If it contains itself, then it doesn't contain itself . If it doesn't contain itself then it does."
