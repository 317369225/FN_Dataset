What is meant by abstract data type in C++?
An abstract data type (ADT) is a collection of data and associated operations for manipulating that data. In simple words it’s a data type that you create to do what you want. Also you should check out the answer to below question for more clarification. What are abstract data types? In C++, a class is a good example of ADT, which can be defined abstract having no public/protected data members (only privates), nor any methods which return pointers/references to any of the private fields.
