What are abstract data types in data structures?
To lessen the complexity of a problem and the problem solving process related to it, the developers use abstraction so as to keep focusing the big picture. An abstract data type comes in handy in this situation. Abbreviated as ADT, it is a logical representation of of we view data and the operations that can performed over them irrespective of the type of implementation. This means that we are concerned only with what the data is representing and not with how it will eventually be constructed. This is how the abstract data type operates.
