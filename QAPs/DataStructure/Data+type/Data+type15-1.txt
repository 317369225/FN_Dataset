Is String data type?
A string is generally understood as a data type and is often implemented as an array of bytes (or words) that stores a sequence of elements, typically characters, using some character encoding. A string may also denote more general arrays or other sequence (or list) data types and structures. String isn't a primitive datatype - it's a class, a reference type. Now admittedly it's supported directly in the VM, and there are literals in the language - but it's still not a primitive type.
