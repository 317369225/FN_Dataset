What is data type in c++?
C++ has many ways to categorize types. However, data type is not usually used as a type category. Here is one categorization: void types (including const/volatile qualifications thereof) function types reference types (including reference-to-function types) object types: everything not in the previous three categories I mention that particular categorization because object types (the most common category) could be equated with what other languages call data types.
