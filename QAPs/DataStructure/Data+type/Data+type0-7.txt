What are abstract data types?
Abstract data type are like user defined data type on which we can perform functions without knowing what is there inside the datatype and how the operations are performed on them . As the information is not exposed its abstracted. eg. Stack and queue are not a data type they are way of implementing a data structure , but they are called as abstract DATA TYPE because we can do operations on datatype only not on function , here in stack and queue we can do operation like pop , push , enqueue etc. and its ABSTRACT because we are not given information about what is there inside the stack or ...
