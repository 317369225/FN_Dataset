What are abstract data types?
Abstract Data Type : A data type can be considered abstract when it is defined in terms of operations on it, and its implementation is hidden (so that we can always replace one implementation with another for, e.g., efficiency reasons, and this will not interfere with anything in the program). An abstract data type or ADT (sometimes called an abstract data type) is a mathematical model of a data structure.
