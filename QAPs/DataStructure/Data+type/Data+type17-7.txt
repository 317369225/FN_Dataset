What is the difference between a data type and a data structure?
Originally Answered: What is the difference between data structure and data type? Data type defines/elaborate the type of data i.e integer, float (real number - with decimal representation), string etc. Data Structure defines how the data will be (efficiently) contained - 1: Data can be stored serially - Stack/Queue/Array. 2: Data can be arranged in tree like structure where the left branch contains The data with value less than that of the parent's & right one is opposite of the left - BST.
