What distributed database supports JSON data type?
MongoDB is one such document store in which there is a query language specific to documents and to json. Joins however are not efficient even if supported, that is why in document stores, the recommendation is to denormalize, i.e. store full json payloads, the way you need them at query time (so you need a processing/ETL phase before inserting to Mongo that restructures JSONs to be read optimized, especially to inline joins data even if duplicating some).
