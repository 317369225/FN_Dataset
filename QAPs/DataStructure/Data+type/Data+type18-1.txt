What is data type in c++?
A datatype actually describes you data like what form of data you wanna create (It may be integer, character, floating point number, string etc.) In the context of C++, you can make many types of data that are mentioned above. To make integer datatype, you type int and then write its name (called variable name). To create floating point datatype , you type float and then write its variable. That is actually a syntax of C++. (It may differ for other languages).
