What are data types in c?
While you are writing a program, you need to have places where you can store values which will be used in your program. These values are stored in the memory and accessed using a variable. Now, data types refer to what kind of a variable you have used. Now data types vary between different programming languages. But generally, we use int for storing integer values. That means, if you need to store an integer, you need to create a variable of type int and then use that to store the value. Similarly for decimal values we use float and characters we use char.
