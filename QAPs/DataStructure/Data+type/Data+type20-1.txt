What means system defined data types and user defined data type?
I would prefer the term language defined data types in place of system defined data types. Well, they are present as part of the language. A programmer just has to use them judiciously. E.g. int, char, float etc in languages like C/C++ and bit, std_logic, real in case of HDLs like VHDL . User defined data types give the programmer the control to define and use them as required. E.g. struct /union (can be a structure of student data/ class data/ shop items etc) and enum ( enum colour={red, green, blue}. Compiler assigns values 0,1 and 2 to red, green, blue resp.
