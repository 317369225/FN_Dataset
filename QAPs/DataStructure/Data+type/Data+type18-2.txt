What is data type in c++?
That’s not a C or C++ issue, but a general aspect of programming languages based on state represented by variables. A “type” is -in it very essence- the set of all the possible values a variable can take plus the set of possible operations that can be done with them. Procedural languages, typically, define a set of “built-in types” that represent the native values the machine they target can manipulate, plus some “aggregation mechanisms”.
