What is the 'virtual' data type in C++? How is it different from other data types?
Virtual isn't a data-type, but you might have thought so because it's often written next to the return type in a function signature: class A { virtual int f(); }; You may have thought that class A contains a member function f which returns a virtual int, but that's not the correct way to read this; rather, class A contains a virtual member function f which returns an int. "Virtualness" is a property of member functions.
