What is the benefit of different data types in programming?
Can you add a cat to a dog? You can’t. Adding is for numbers, not animals. Can you breed 2 and 3? You can’t. Breeding is for animals and plants. Can you divide an angle by a date? A date by an angle? A data type only allows some values. A type about animals won’t allow ‘1’ as a valid value. A data type only allows certain things to done with values. These may be standard stuff (e.g. arithemetic rules) or functions you specify (e.g. a routine for breeding animals).
