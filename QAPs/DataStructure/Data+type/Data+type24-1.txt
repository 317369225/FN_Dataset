Is an Array an abstract data type?
I’d say according to the conventional thinking, no it is not abstract. When I think of an array I think of a homogeneous collection of a given type stored in contiguous blocks of memory wherein individual elements can be accessed in O(1) time based on an index that represents an offset from the base address of the array. In this sense, no, array is not an abstract data type — I’ve defined too many implementation details.
