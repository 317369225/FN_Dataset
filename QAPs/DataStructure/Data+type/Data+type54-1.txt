Why data types are required in any programming language?
Data types are required so that you know what "kind" of data your variable holds, or can hold. If you (and compiler, and runtime and what not) know this information in advance, you can save a hell lot of runtime issues at compile time only. For example, in Java, when you declare an int variable, compiler and JVM knows that this variable is going to hold only integer values, nothing else. But when you declare a variable in Javascript using var, you are just declaring a name. This variable can hold any value, be it a primitive type or a complex object.
