What is data type in Java?
Data types represent the different values to be stored in the variable. In java, there are two types of data types: Primitive data types : Primitive data types are those whose variables allows us to store only one value but they never allows us to store multiple values of same type. This is a data type whose variable can hold maximum one value at a time. Non-primitive data types : Non-primitive data types are those whose variables allow us to store multiple values of same type.
