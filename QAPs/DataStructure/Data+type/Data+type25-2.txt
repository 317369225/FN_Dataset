What is the difference between a data type and an abstract data type (ADT)?
Thanks for the A2A. A data type is a way of telling the compiler or machine, about what valid values a particular variable can hold. Also , user-defined data types like class-types also define the various behavioral aspects of that object. For example consider the following statement. int a = 1; Here, a is an int datatype. It can only hold integral values. If we attempt to initialize it with values like true/false, 0.001, 2.34, it fails, or will not be properly initialized. A data type is a concrete concept.
