Is array a data type or data structure?
Data structure : Organizing data for specific purpose . Purpose can be anything like reducing time complexity , last inserted data should come first (Browser Click to go back button) , first inserted data should come first ( Bank service) etc etc. For our case if I insert collection of homogeneous values into array , I can access specific index with O(1) time complexity. So array is data structure :) :) . In array we arrange values in sequential manner . Hence array called linear data structure .
