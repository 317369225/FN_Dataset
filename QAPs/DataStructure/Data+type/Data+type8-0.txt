What's the difference between a data structure and an abstract data type?
An abstract data type is a specification (or interface) for how to interact with a sort of data structure which doesnâ€™t make any statements about how the data structure works. So a list can be described in terms of an abstract data type (we can insert into it, get the nth element, delete an element, etc.), while a linked list is an implementation of that abstract data type (it implements the specified behavior by structuring the data as either an empty list or a pairing of a data element and another linked list).
