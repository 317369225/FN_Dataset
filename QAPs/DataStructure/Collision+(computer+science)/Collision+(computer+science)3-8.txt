What are some good computer science final year project ideas?
Originally Answered: What are some project topics for my final year project in computer science?Do a project on something that interests you. There are so many topics to start exploring. You can find great projects on Data Mining , Cloud Computing . You can try building a web framework tool. If you want to build Android application try building some battery optimizer or a benchmark tool. You can try working in building a secure system. You can also try implementing an image processing system. There is a lot of choices available. It is on you what interest you and what you would like to pursue. All the best !
