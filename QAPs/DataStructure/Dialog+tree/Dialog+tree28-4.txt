Which is the best language, C or C++?
All depends on what you want to program. If you want to write a program which is the fastest possible then the C language would be the right choice. For example some programs that manage the video must absolutely be written in C or better in assembler. Otherwise the C++ is much more comfortable to use. The classes facilitate the manipulation of complex structures. For example the management of lists, trees, tables, dialog boxes, windows. With C language you must always manipulate pointers, pointers to pointers,...
