What is type wrapper in Java?
A wrapper type/class in the Java programming language is one of eight classes provided in the java.lang package to create objects for the eight primitives types. Wrapper classes are used to represent primitive values when an Object is required. The Wrapper classes are:  1.  Byte  2.  Short  3.  Integer  4.  Long  5.  Float  6. Double  7.  Character  8.  Boolean The names of these wrapper classes are just the same as their primitive data type with a capitalized first letter. These wrapper classes start with a upper case alphabet while their primitive counterparts start with a lowercase alphabet. The wrapper classes are used extensively with Collection classes in the java.util package and with the classes in the java.lang.reflect reflection package.  Example: ArrayList<Integer> arr = new ArrayList<Integer>();arr.add(5); // 5 here is an int but java does auto-boxing converts this int to Integer object and then adds it to arraylist.
