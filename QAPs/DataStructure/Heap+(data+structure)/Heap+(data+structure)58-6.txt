What basic data structures and algorithms should one learn before starting competitive programming?
Sorting is the most heavily studied concept in Computer Science. Idea is to arrange the items of a list in a specific order. Though every major programming language has built-in sorting libraries, it comes in handy if you know how they work. Depending upon requirement you may want to use any of these. Merge Sort Quick Sort Bucket Sort Heap Sort Counting Sort More importantly one should know to use them.
