What are the main data structures layman's explanation (Array, Stack / Queue, Hashset / Hashmap / Hashtable / Dictionary, Tree / binary tree, Heap, Graph)?
There are three basic ordered data structures: Set, Tree and Dictionary. Sets are ordered sets, trees are also ordered sets but provide searches by a separate key type and dictionaries are ordered sequences of Key/Value pairs (often called an associative array). An Array is a special case of a dictionary (i.e. Array<T> is somewhat similar to Dictionary<long,T>). Arrays also serve the purposes of lists and stacks.
