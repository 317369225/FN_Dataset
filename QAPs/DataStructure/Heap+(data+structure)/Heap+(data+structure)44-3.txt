How do I start learning or strengthen my knowledge of data structures and algorithms?
I'm a big fan of learning by doing.  If you already know the basics of a language, then start looking around for a project to contribute to.  The project might be a simple web site, an special-purpose search index, or a set of simple application prototypes.  If you can find a real open source project, that you can contribute to in a meaningful way, that's even better.  Increasingly, companies are even considering open source contributions as a factor in recruiting and hiring.
