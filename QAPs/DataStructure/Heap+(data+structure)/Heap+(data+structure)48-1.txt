What are the best books on algorithms and data structures?
The CLRS book has been listed as the textbook in the large undergraduate course on algorithms that I taught for the last two semesters. To my surprise, many students find it unreadable and not helpful when working on practical projects because it does not use a realistic programming language. For a beginner, I would recommend Algorithms by Dasgupta, Papadimitriou, and Vazirani. If you are interested in using practical languages, I recommend the book by Mark Allen Weiss "Data Structures and Algorithm Analysis in C++" (4th edition). It uses C++11.
