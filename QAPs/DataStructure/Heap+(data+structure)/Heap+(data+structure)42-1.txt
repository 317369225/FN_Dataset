How tough is it to clear the bridge courses for CS at RIT?
If you start studying in Jan or in Feb you can waive CSCI 603 and CSCI 661 for sure. If you have some experience(1 year) in Java then you can waive the CSCI 605. I would say focus on only one course as one have to score 80%+ in order to waive a course. If you focus on CSCI603 (my favorite) then start with 3 simple one-month Coursera courses.
