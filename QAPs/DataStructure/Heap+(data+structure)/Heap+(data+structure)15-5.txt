What are the most important data structures in a tech interview?
Most important data structure in a tech interview is Array. I have appeared for many interviews and seen that array questions are always asked. The interviewer will give you a very simple array question initially, for example - binary search in a sorted rotated array, just to check if you can handle all the edge cases, to see what is your coding style and basically, to check if you can actually write code so that the interview process can be carried forward further.
