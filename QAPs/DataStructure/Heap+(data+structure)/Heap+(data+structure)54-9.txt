What algorithms and data structures should any software engineer know?
If you’re preparing for an interview or generally to learn data structures to help you in writing code, here are my recommended preparation steps: Strong fundamentals: You can use Introduction to Algorithms by Cormen. Focus on these chapters: Sorting Stacks & queues Trees (you can avoid AVL, Red-Black Trees) Graphs Dynamic and Greedy programs Get your hands dirty: Don’t just look at algorithms. You need to pick one programming language that you are comfortable with and start writing code using an IDE (say Eclipse) for that language.
