What are the most important data structures in a tech interview?
Array, linked list, doubly linked list, AVL and/or red-black binary search tree, hash table, B-tree.  Interviewee should know, for each, types of insertions/deletions/searches and time complexity of each.  Also should know memory overhead for these data structures.  Should know which/when a certain data structure should be used to implement a stack or a queue.  Should know quicksort and heapsort for arrays.
