What is the simplest C++ code for Prim's MST algorithm?
Here is Prim with complexity O(N^2): // This is the distance array memset(d, 0x7f, sizeof d); // set all elements to some big values d[0] = 0; // This array keeps the vertex corresponding to the value of d memset(trace, 0, sizeof trace); // we repeat N-1 times, each time select 1 edge for (int turn = 0; turn < N-1; ++turn) {     // select vertex with minimum distance     int u = min_element(d, d+N) - d;     // We found 1 more edge: (u, trace[u])     // update distance to other vertex     for (int v = 0; v < N; ++v)     if (d[v] > cost[u][v]) { // cost is the matrix that stores cost of edges.         d[v] = cost[u][v];         trace[v] = u;     } }
