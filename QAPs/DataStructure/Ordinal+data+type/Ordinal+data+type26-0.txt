Which online Python course is the best?
In my humble opinion, "Learn Python the Hard Way" by Zed Shaw is a fantastic way to learn Python. It has a bunch of brilliant exercises which teach the fundamentals of the language very well. Zed being a very accomplished programmer and writer himself has really nailed it. The book is also free (gratis). Highly recommended - http://learnpythonthehardway.org/ Also recommended "A Byte of Python" by User - http://www.swaroopch.com/notes/P...
