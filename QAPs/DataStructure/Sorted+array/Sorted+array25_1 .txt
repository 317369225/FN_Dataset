In a sorted array is it possible to find two numbers in the array having difference equal to a given number in O(N)?Adding to Tarundeep Singh's answer. Even if the array is unsorted, you can get it in O(n) time. Let the given difference be K. Also, let the sorted array be A[0 to n-1]  1 2 3 4 5 6 7 8 91011121314151617 if(A[n-1]-A[0] < K)  Not Possible else {  for(int i=0;i<n-1;i++) //O(n)  {  hash of (A[i]) =1;  }  for(int i=0;i<n-1;i++) //O(n)  {  if(hash of (A[i]+K) ==1)  {  FOUND;  break;  }  } } This code will take O(n) execution time .
