What tools do data scientists use for professional presentations (e.g. diagrams, graphs, flow charts, etc.)?
I like to use The IPython Notebook. I recommend setting up an iPython Notebook server so you can easily work with someone on a report. Another benefit is that you can share a link to a report, and if you update the report, the link is still valid. For interactive charting on web dashboards I like to use NVD3. It's a convenient out-of-the-box charting library that doesn't require one to be far along in D3 or JavaScript.
