What tools do data scientists use for professional presentations (e.g. diagrams, graphs, flow charts, etc.)?
My answer overlaps a lot with the existing answers. To create flow-charts or simple diagrams/graphics, I have been using Keynote and it hasn't failed me. Keynote is limited in what it's able to do. But for what it covers, it's the best time-saving tool I can find out there. To create data visualizations I use either R-ggplot2 or Tableau, depending on the workflow that I was in. Both of these are fantastic tools for visualizing data, where ggplot2 is more statistically focused and Tableau is more business intelligence-like.
