What tools do data scientists use for professional presentations (e.g. diagrams, graphs, flow charts, etc.)?

For network (graph) visualizations I typically use Cytoscape (for which I've written various plug-ins for specific things), but another group has recently licensed some HTML5-based network/graph toolkits that I'll probably use.
