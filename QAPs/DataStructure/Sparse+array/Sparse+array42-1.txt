What are some important tips and information that one should know when transitioning from jQuery to native JS (DOM manipulation)?
jQuery is so nicely written library - Using Pure native JavaScript API's. You can walk through source code to understand how native API's work. E.g - This is how your jQuery.inArray is written function(elem, arr, i) {   var len;   if (arr) {     if (indexOf) {       return indexOf.call(arr, elem, i);     }     len = arr.length;     i = i ? i < 0 ? Math.max(0, len + i) : i : 0;     for (; i < len; i++) {       // Skip accessing in sparse arrays       if (i in arr && arr[i] === elem) {         return i;       }     }   }   return -1; }
