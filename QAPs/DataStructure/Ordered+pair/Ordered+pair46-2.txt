What is an integer?
The integers can be defined as the set of equivalence classes of ordered pairs of natural numbers under the following relation: [math](a,b) \sim (c,d)[/math] If [math]a+d = b+c[/math] Addition is defined as: [math](a,b)+(c,d) = (a+c,b+d)[/math] Additive inverse is defined as: [math]-(a,b) = (b,a)[/math] Multiplication is defined as [math](a,b)\times(c,d) = (ac+bd,ad+bc)[/math] An integer can be represented as: [math](a,b) =[/math] [math]a-b[/math] if [math]a>b[/math] [math]0[/math] if [math]a=b[/math] [math]-(b-a)[/math] if [math]a<b[/math] The integers are a commutative ring.
