Can every real number be expressed in the form of x^x, where x can be any real number?
This function  f(x)=x x     is continuous on the interval  [0,∞)    and takes its smallest value at  x=1/e.  That smallest value is  f(1/e)=e −1/e     which is about 0.6922. Therefore, every number greater than or equal to  e −1/e     is equal to  x x     for some number x.  Addendum By the way, as written, f(x)=x x     isn't defined at  x=0.    It does have a limit, though, as  x→0,  and that limit turns out to be 1, which you can see from the graph, so it can be extended continuously to the domain  [0,∞).  Here's a formal proof that the limit is 1.  First note that  x x     can be written as  e xlogx     where  log  denotes the natural logarithm.  We'll show that                                 lim x→0 (xlogx)=0  and it will follow that                                lim x→0 x x =lim x→0 e xlogx =e 0 =1.  But why does  lim x→0 (xlogx)    equal 0?  You can rewrite this limit as  lim x→0 logx1/x .    Since this limit is of the indeterminate form  ∞∞ ,  we can apply L'Hôpital's rule.  The limit will be the same as the limit of the quotient of the derivatives of the numerator and denominator:                               lim x→0 logx1/x =lim x→0 1/x−1/x 2    which equals  lim x→0 −x=0.  Updated 5 Sep • View Upvotes
