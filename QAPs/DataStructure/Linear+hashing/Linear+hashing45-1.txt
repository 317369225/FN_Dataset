How do I count the number of subarrays (contiguous) with sum zero using DP?
The brute force solution would be to consider all the sub arrays and check the sum of each sub array. There would be an outer loop at the starting point i and an inner loop which will try all the sub arrays starting from i. The complexity of this solution would be O(n^2), so you might need to optimize it. Hashing would be a wonderful approach to do that. You would need to iterate through the array for every element a[i] and calculate the sum from 0 to i. If the current sum has been seen before, there is a zero sum array. Hashing is used to store the sums to check whether the current sum is seen before or not. import java.util.HashMap;   class ZeroSumSubarray {          // Returns true if arr[] has a subarray with sero sum     static Boolean printZeroSumSubarray(int arr[])     {         // Creates an empty hashMap hM         HashMap<Integer, Integer> hM = new HashMap<Integer, Integer>();                  // Initialize sum of elements         int sum = 0;                         // Traverse through the given array         for (int i = 0; i < arr.length; i++)         {               // Add current element to sum             sum += arr[i];                          // Return true in following cases             // a) Current element is 0             // b) sum of elements from 0 to i is 0             // c) sum is already present in hash map             if (arr[i] == 0 || sum == 0 || hM.get(sum) != null)                                           return true;                          // Add sum to hash map             hM.put(sum, i);         }                     // We reach here only when there is no subarray with 0 sum         return false;     }                 public static void main(String arg[])     {         int arr[] = {4, 2, -3, 1, 6};         if (printZeroSumSubarray(arr))             System.out.println("Found a subarray with 0 sum");         else             System.out.println("No Subarray with 0 sum");                }            } Source: Find if there is a subarray with 0 sum - GeeksforGeeks
