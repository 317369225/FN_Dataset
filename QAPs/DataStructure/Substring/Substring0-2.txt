Write a program to return the longest repeating  substring in a string.
We can use KMP algorithm to find periodicity of strings. The following method describes a way to find the length of the period. Let the length of the string be [math]n[/math]. Build the failure function table [math]T[0...n][/math] using KMP algorithm (this is the preprocessing step of KMP) [1][2]. Return [math]n - T[n] [/math] To check if the string is periodic, check if [math]n[/math] is divisible by [math]n - T[n][/math]. If yes, then the string is periodic.
