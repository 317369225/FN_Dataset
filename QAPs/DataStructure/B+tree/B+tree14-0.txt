What is the simple possible application using B tree?
The simplest reasonably appropriate use would be for recording and checking a list of keys. Imagine a spellcheck: You want to read a list of correct words, and then check every word typed against those words. Storing all the words in a tree of some sort makes the lookups quicker - scanning through an entire list each time would be slow (scaling with the number of words), while a looksup in a tree scales with the logarithm of the number - roughly speaking.
