What is B* Tree in Data Structure?
The aim of a binary tree is to half the records remaining with each search step. To do this we have some sort of value to check. If it is the exact match than we found the record. If the value is higher than the record we started with then we follow the higher pointer. If there is no high pointer then we report not found.
