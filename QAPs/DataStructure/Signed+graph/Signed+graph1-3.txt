Is it possible to solve the equation [math]x^3 - 6 x^2 + 3x + 10 = 0[/math] without the help of the hit-and-trial method?
Let's try Cardan's formula (if you really want a general, no trial and error, method). Cubic function Given [math] f(x) = x^3+ax^2+bx+c =0 [/math]. Firstly remove the [math] x^2 [/math] term by applying [math] x = y-\dfrac{a}{3} [/math], we have [math]  f(x) = f(y-\dfrac{a}{3}) = y^3 + py +q = 0 [/math], where [math] p = b-\dfrac{a^2}{3} [/math], [math] q = c - \dfrac{ba}{3}+\dfrac{2a^3}{27} [/math]. Then we can find three [math] y [/math] solutions to be, [math] y_1=\sqrt[3]{A}+\sqrt[3]{B} [/math], [math] y_2=\omega\sqrt[3]{A}+\omega^2\sqrt[3]{B} [/math], [math] y_3=\omega^2\sqrt[3]{A}+\omega\sqrt[3]{B} [/math], where [math] A = -\dfrac{q}{2} + \sqrt{\dfrac{q^2}{4}+\dfrac{p^3}{27} }[/math], [math] B = -\dfrac{q}{2} - \sqrt{\dfrac{q^2}{4}+\dfrac{p^3}{27} }[/math], [math] \sqrt[3]{A}\cdot\sqrt[3]{B} = -\dfrac{p}{3} [/math], [math] \omega = \dfrac{-1+i\sqrt{3}}{2} [/math]. For this problem [math] a = -6,\,\, b = 3,\,\,c = 10 [/math]. We can find [math] p = -9,\,\, q = 0 [/math], [math] A = i\sqrt{3^3},\,\,B =-i\sqrt{3^3} [/math],  [math] \sqrt[3]{A} = -i\sqrt{3},\,\,\sqrt[3]{B} =i\sqrt{3} [/math], and [math] \{y_1 = 0, y_2 = 3, y_3 = -3\} [/math]. Thus [math] \{x_1 = 2, x_2 = 5, x_3 = -1\}  [/math], [math] x^3-6x^2+3x+10 = (x-2)(x-5)(x+1) = 0 [/math].
