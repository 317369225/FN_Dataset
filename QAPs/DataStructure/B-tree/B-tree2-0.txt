What is a B+ Tree?
A B+ tree is a data structure often used in the implementation of database indexes.  Each node of the tree contains an ordered list of keys and pointers to lower level nodes in the tree.  These pointers can be thought of as being between each of the keys.  To search for or insert an element into the tree, one loads up the root node, finds the adjacent keys that the searched-for value is between, and follows the corresponding pointer to the next node in the tree.  Recursing eventually leads to the desired value or the conclusion that the value is not present.
