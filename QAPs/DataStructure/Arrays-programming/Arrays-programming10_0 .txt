Why can't constant values be used to define an array's initial size?No, Constant value can used to initialize array size. Please refer following example. #include <stdio.h>  int main() { const int n=5; int arr[n],i; for(i=0;i<n;++i) arr[i]=i; for(i=0;i<n;++i) printf("\n %d",arr[i]); return 0; } This program compile without any error and produce output as 0 1 2 3 4 34 Views
