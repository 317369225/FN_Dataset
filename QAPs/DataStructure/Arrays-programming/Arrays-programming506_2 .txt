How do I load various Arrays in the same TableViewController in SWIFT?I want to load a different array into my TableViewController once a certain button is pressed. E.g.: breakfast, dinner, lunch etc... I have no idea how my tableviewcontroller.swift file can recognize a button being pressed...Does anyone have a idea or demo file somewhere for me too look at?Link all buttons to show the view controller with the table view, because every buttons is going to do that. In your first file (ViewController), have this code: var contentMode = 0 @IBAction func button1(sender: AnyObject) { contentMode = 1 } @IBAction func button1(sender: AnyObject) { contentMode = 2 } And in your second file (TableView), have this code: let array1 = ["Cow", "Horse", "Pig"] let array2 = ["Dog", "Cat", "Hamster"] func tableView(tableView: UITableView, numberOfRowsInSection section: Int) -> Int { if ViewController().contentMode == 1 { return array1.count } if ViewController().contentMode == 2 { return array2.count } } func tableView(tableView: UITableView, cellForRowAtIndexPath indexPath: NSIndexPath) -> UITableViewCell { let cell: UITableViewCell = tableView.dequeueReusableCellWithIdentifier("Cell")! as UITableViewCell if ViewController().contentMode == 1 { cell.textLabel?.text = array1[indexPath.row] } if ViewController().contentMode == 2 { cell.textLabel?.text = array2[indexPath.row] } } I hope this makes sense, this is how I would do it.
