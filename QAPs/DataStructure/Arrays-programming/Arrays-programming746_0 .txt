How can I post an array of js object to struts2 action via jquery?You can use AJAX to do this. Something along the line of this would do: $.ajax({ type: "GET/POST", url: "myAction", //pass any query like myAction?param=value data : {myJSONString}, //convert your JS Object to JSON first using JSON.stringify(myJSObject) dataType:"json", contentType: "application/json; charset=utf-8", success: function(data,status,xhr) { alert(xhr.responseText); } error : function(xhr) { alert(xhr.status); } }); Hope this will do the trick.
