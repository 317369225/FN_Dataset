Are arrays objects in Java?The Java Language Specification section 4.3.1 starts off with: An object is a class instance or an array. Yes; the Java Language Specification writes: In the Java programming language arrays are objects (4.3.1), are dynamically created, and may be assigned to variables of type Object (4.3.2). All methods of class Object may be invoked on an array. 230 Views  View Upvotes
