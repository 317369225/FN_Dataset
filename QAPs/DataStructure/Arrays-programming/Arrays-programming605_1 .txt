What will be maximum subscript of array having size?There can be 2 cases here: Most programming languages, like Google Go & C, have indexes starting from 0 to size-1. So, the maximum subscript/index is N-1. Array element is accessed for example as A[N-1] Few programming languages, like Visual Basic, have indexes starting from 1 to N. So, maximum subscript/index is N. Array element is accessed for example as A[N] Best of luck 36 Views  Answer requested by Kandala.vamshidhar Reddy
