What's the name of the feature that allows PHP to use an array element to call a method?Like this: $playerModel->$array['element']($input); ?The feature that makes this possible in PHP is called Variable Functions. See Variable functions Basically, the array element has to evaluate to the name of a function that PHP can find within the runtime context of the call.
