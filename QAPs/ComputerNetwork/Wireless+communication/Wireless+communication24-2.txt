★Wireless Communication: What are the best books to learn about the Physical layer implementation of LTE?
Understanding LTE with Matlab -by Houman Zarrinkoub : This book contains concise explanations for the PHY layer blocks with helpful code snippets that you can simulate and understand the concepts better. 4G: LTE/LTE-Advanced for Mobile Broadband -by Erik Dahlmann:  This book not only gives the implementation details but also explains the whys of LTE. For e.g. why a certain type of equalizer is used over others etc. This helps understand the concepts in a very simple way.    Embed Quote
