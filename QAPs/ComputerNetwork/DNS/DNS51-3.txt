★What is the difference between URL Forwarding and attaching with DNS?One is URL forwarding to any other website and another means to attach directly with server using DNS settings.. Is there any difference? If not then why there are two different terms?
URL forwarding or domain forwarding are one and the same. URL forwarding is redirection or redirecting url ‘A’ to url ‘B’ whereas DNS is domain name system which enables this redirection on server level via the internet. DNS has multiple records types like A, AAAA, cname, SRV, TXT, SOA, MX.. Usually URL/domain forwarding are achieved using an A record.
