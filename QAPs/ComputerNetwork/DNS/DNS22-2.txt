★How does Amazon's Route 53 compare to Google Public DNS and OpenDNS?
Google Public DNS is a caching and recursive DNS name server offering a name resolution service typically use by end user clients, OpenDNS also appears to be a caching and recursive DNS name server offering with some additional offerings that may change the response based on policy.
