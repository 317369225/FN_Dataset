★What IP address does a client use as source IP address while making a DHCP request and where does the DHCP server send the reply to?
Host >>>> dhcp server Discovery frame is sent from host to any DHCP server in an FFFF.FFFF.FFFF.FFFF broadcast DHCP server responds with an OFFER frame and the source of its mac address Host responds with a REQUEST frame using the destination MAC of the DHCP    At this point the host has an IP address, subnet mask and default gateway and maybe domain name. DHCP server responds with a ACKNOWLEDGE frame and adds the host ip address and mac to its dhcp database. For short. The process is DORA Discovery > host Offer > dhcp server Request > host Acknowledge > dhcp server
