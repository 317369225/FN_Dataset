★What is the difference in architecture between the MySQL and Tomcat application server?In other words, why cant some modification of tomcat be used at both application and database layers ?
You could, and for some very small and specific applications you could certainly use Tomcat (probably just build and deploy a WAR rather than modifying its source code) to serve as a key value pair data store or even a file retrieval system.  Having said that, I could also try and use a Mini Cooper as a work truck and while it would function to some extent it wouldn't be particularly good at because it was designed for other tasks. MySQL is a Relational database management system Tomcat is an Application server
