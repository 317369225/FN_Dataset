★What is the best software for managing a music collection over a home network?Requirements: *I can manually manage my folder/file structure *New files are automatically added to library *Can add files to your local mobile device from the network library.
If you're running windows then MediaMonkey is the way to go.  It supports just about any mobile device, has a built in DLNA server so you can listen to music or watch videos over any client over the network and has some of the best organization tools out there.  I wrote up a summary of the top ones here:  Digital Music Management with Media Monkey Part 1
