★What is wireless sensor network security and what is its purpose?
Wireless  Sensor  Networks  Security  means  secure  the  data  from   any  kind  of  attack  from  outside  or  inside  of  the  network.  This  network is  always    exchange  small  foot  print  data   in     like  Sensor  reading, Configuration  details,Operational  data..  and  so  on.  Security  is  always  depend  on  what  kind  of  application  really  looking  in  to ,  and  this  is  really  necessary  in  some  unexceptional  cases  like ,  Radiation  Monitoring, Disaster  Management,  Water  Quality  Management, Fire  Alarm.  If  we  ready  to  compromise  the  security in  these  kind  of  application ,  then  its  a  waste  to  implement  the  WSN    for  any  application.  Security  will   provide  the protection    for  the  device  from  inside  and  outside attack Inside  : Fake  device  Attack, Continuous  Beacon  Attack,Denial  of  Service Attack,....... Outside :  Physical  Attack  on  Device .(Providing  Security  to  this  attack  is   really  challenging  for  WSN)    Embed Quote
