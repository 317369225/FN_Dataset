★What are the complications we might face while developing very large scale (1000 nodes) wireless sensor network?
Data  Management:         Data  Management  is  an  issue  While  implementing  a  very  large  network, Here  we  had  done  one  test  on 65536  nodes  in  one  network. then  we  facing  one  problem  in  managing  large  amount  of  data, if  65000 is  runs continuously  up  to   5  hours ,  each   2 s  it  generate  nearly  125 MB  of  data(Packet) . Packet  Delay may Happens     Packet  Delay  is  happens  due  to  multiple  protocol were  working  in the  same  field (ZigBee, Bluetooth, Wifi). even  if  they  were  using  different  protocol  all  the  three  were  using  the same  frequency  and  two of  them  were using same  MAC  layer  protocol  IEEE  802.15.(Becon  enabled Networks) Drain  Power of  Low  Power Device.    All  the  Sensor  Network  device  are  low-power  device. when  we  concern  about  the    very  large   network  due to  broadcast beacon  from  1000  device  or  more than  1000    make   a    power drain  from  its neighbouring  nodes.    Embed Quote
