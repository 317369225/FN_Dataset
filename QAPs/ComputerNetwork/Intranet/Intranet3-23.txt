★What are Sharepoint alternatives for a company-wide intranet?In a ~500-1000 person company, what alternatives are there for a  document, file repository-style intranet portal? Bonus points if there is room and potential for other elements of the information workplace (such as collaboration, social, content management) down the road.
You can take a look at Stackfield! (Social Intranet for enterprises) This cloud collaboration tool combines real-time chat with task management, file sharing and scheduling. You can simply create new workspaces, which can be adapted individually, for each need. All relevant data are end-to-end encrypted directly in your browser, therefore it's the most secure intranet in the cloud.
