★What are Sharepoint alternatives for a company-wide intranet?In a ~500-1000 person company, what alternatives are there for a  document, file repository-style intranet portal? Bonus points if there is room and potential for other elements of the information workplace (such as collaboration, social, content management) down the road.
Interact Intranet have a Microsoft based product, but you could also use a few smaller open source CMS. You might also check out MangoSpring o if you want a fully social internal communication tool that you can integrate in an intranet. Check out this site for some inspiration for social software that you can deploy enterprise wide: http://socialsoftwarematrix.org/
