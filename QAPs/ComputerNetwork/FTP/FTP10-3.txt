★What is the best way to install FTP on an Amazon EC2 server?
Here's a stripped-down vsftpd sample configuration for use on Ubuntu and EC2 (with SSL, active and passive mode, virtual users and chroot. Please not that $IP should be replaced with your AWS Elastic IP address. If you find any optimizations, please let me know: listen=YES ssl_enable=YES require_ssl_reuse=NO force_local_data_ssl=YES force_local_logins_ssl=YES rsa_cert_file=/etc/vsftpd/vsftpd.pem  ftpd_banner=Hello anonymous_enable=NO local_enable=YES virtual_use_local_privs=YES write_enable=YES port_enable=YES port_promiscuous=YES connect_from_port_20=YES secure_chroot_dir=/var/run/vsftpd pam_service_name=vsftpd guest_enable=YES user_sub_token=$USER local_root=/var/ftp/$USER chroot_local_user=YES hide_ids=YES local_umask=0022  # logging syslog_enable=YES xferlog_enable=YES  # passive mode pasv_address=$IP pasv_min_port=20000 pasv_max_port=21000 You will also need to extend PAM config by adding following data to /etc/pam.d/vsftpd: auth    required pam_pwdfile.so pwdfile /etc/vsftpd/passwd account required pam_permit.so
