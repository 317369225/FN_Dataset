★What is the best way to install FTP on an Amazon EC2 server?
Don't. Use HTTPS file uploads, it's secure and you won't have all the port headaches.  Or at least use sftp instead. Failing that, Install FTP however you normally would for the given OS (if you're doing Linux, then via yum or whatnot) Open port 21 in the Amazon security group to your instance from whatever IP addresses you want to have access Decide if you want active or passive FTP, then open more ports to fit.There's a really good walkthrough here: http://linuxadminzone.com/instal...
