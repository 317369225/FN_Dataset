No, this analogy is wrong because the domain name system is distributed. Each level (the dots in the domain name) is a level in the hierarchy. Alan Cohen's answer is the very top level. At each lower level, the authority for the host name is delegated to another level of nameservers. For example, for Quora.com, the top level would be the IANA root servers. The next would be .com servers, operated by VeriSign (historically, Network Solutions which was acquired by Versisign.) Some of these IP addresses are anycast, so there are multiple sites sharing the same IP, for geographical distribution.  1
