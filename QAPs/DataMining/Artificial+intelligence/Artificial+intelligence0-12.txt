SimpleCV - http://simplecv.org/ Make computers see with SimpleCV, the Open Source Framework for Vision. SimpleCV is a framework for Open Source Machine Vision, using OpenCV and the Python programming language. SimpleCV provides Machine Learning with KNN, NaiveBayes, SVM and Tree Classifiers. Apart from Template Matching, Feature Extraction, Face Recognition, it also has Tracking support. It has its own IPython shell(something like MATLAB). And an Android application.    Embed Quote
