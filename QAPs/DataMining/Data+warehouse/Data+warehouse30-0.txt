★What are some good examples of applications written against data warehouses?
Most commonly I see applications that use the data warehouse as a batch processing tool for pre-calculating certain application features.  For example, I've seen this done for social networking in order to identify people you might know [1][3].  This is also similar to how Amazon does its recommendation engine for all those fun products we browse [2].
