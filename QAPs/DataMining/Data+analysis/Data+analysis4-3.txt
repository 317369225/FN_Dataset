Which are some of the best Data Analysis tools?
Depending on your needs and your expertise in statistics and programming, there are very different options: If you are an expert, try SPSS, Wincross, Q Professional  or R If you only want something very basic and simple, most data collection software have some data analysis features. Check this page for a summary: Choosing Survey Analysis Software If you want something more than just basic data analysis but don’t have time to invest in knowing the software in option 1, try DataCracker (website: DataCracker - Find the insights hidden in your survey data). You upload your data, it runs statistical tests and gives you a basic report which highlights interesting results from the data. From there you can do further analysis by creating new variables from existing responses, or using word cloud, segmentation and predictive modelling.    Embed Quote
