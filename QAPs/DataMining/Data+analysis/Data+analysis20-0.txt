What's the best free data analysis and visualization software out there right now?
For data visualization, I'd recommend checking out: Prefuse (http://prefuse.org/) Flare (http://flare.prefuse.org/) Processing (http://processing.org/) Protovis (http://vis.stanford.edu/protovis/) RaphaelJS (http://raphaeljs.com/) ProcessingJS (http://processingjs.org/) These are visualization libraries that you can use to do in-depth analysis on your data. If you're looking for canned programs, check out: Gephi (http://gephi.org/) Many Eyes (http://manyeyes.alphaworks.ibm.c...)
