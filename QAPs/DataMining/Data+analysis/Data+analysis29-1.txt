What are some real-time data analysis frameworks?
WSO2 Complex Event Processor  (wso2.com/products/complex-event-processor/) is a Complex event processing engine that accepts realtime analytics queries as SQL like queries. You can run it as a single node or graph of CEP nodes running on top of Apache Storm. We have processed up to about 0.8 million events per second using this, and it can do 100K+ on a single node.
