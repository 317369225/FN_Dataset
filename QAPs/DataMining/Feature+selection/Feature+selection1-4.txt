★How do I perform feature selection?
PCA is not a feature selection technique, but one may consider it as a a feature extraction technique. feature selection means selecting a useful subset from all features. So you must not transform input features into another set of features  in feature selection. One technique to select features is using some ranking criteria to rank features for example: 1) Correlation Criteria: with this criteria you can rank features in order of their correlation with the labels of the data. 2)Mutual Information Criteria: high mutual information of a feature with the label means high relevance of that feature. Another technique is Wrapper methods. you can read more about feature selection in the following article: Guyon, I., & Elisseeff, A. (2003). An introduction to variable and feature selection. The Journal of Machine Learning Research, 3, 1157-1182.    Embed Quote
