★What are some good learning projects to teach oneself about machine learning?This is a follow-up question to How do I learn machine learning?.
HackerRank has quite a few Machine Learning challenges. There is an entire track onHackerRank dedicated to 'Machine Learning'. They also organize Machine Learning contests frequently. Quora CodeSprints are one of those contests. The training data for the challenges (Quora Answer Classifier) is sampled from the real production data of Quora.
