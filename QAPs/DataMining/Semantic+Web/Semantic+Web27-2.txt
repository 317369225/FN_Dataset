★What Semantic Web programming frameworks exist besides Jena?
Sesame - www.openrdf.org Sesame is an open source framework for storage, inferencing and querying of RDF data. Sesame matches the features of Jena with the availability of a connection API, inferencing support, availability of a web server and SPARQL endpoint. Like Jena SDB it provides support for multiple backends like MySQL and Postsgre. Sesame Native is the native triple store offering from Sesame. As compared to be Jena’s native triple, TDB, it is less scabable. Sesame native has been tested with up to ~70 Million triples while Jena TDB is known to work with 1.7 Billion triples.    Embed Quote
