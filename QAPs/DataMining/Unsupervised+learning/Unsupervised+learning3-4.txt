★What is the difference between supervised and unsupervised learning algorithms?
A supervised learning algorithm knows about the kind of data that it deals with as it is trained to do so. Hence such algorithm can be used to classify data according to its category. Now in unsupervised learning algorithm the data is not known before hand and has to be classified using its peculiarities or common traits.
