★What is Dublin Core metadata?
When people say "Dublin Core", they usually mean one of two different (but related) things.  Sometimes they actually mean the Dublin Core Metadata Initiative (or DCMI for short).  DCMI has a webpage at http://dublincore.org.  However, they are more frequently talking about the Dublin Core Element Set which is a small vocabulary, or set of terms, that are useful for general description of things (i.e., metadata) in databases and on the web.  DCMI is the open community that created the Dublin Core elements starting back in 1995.  You can learn more about DCMI and Dublin Core Metadata by poking around on the website noted above; or, you can go straight to the page there the metadata terms are defined at http://dublincore.org/documents/....    Embed Quote
