★How can I read PDF metadata from the command line?Is there a way to extract the metadata from a pdf file using the linux command  line?
pdfinfo will print the contents of the 'Info' dictionary plus some other useful information. The 'Info' dictionary contains the following values: title subject keywords author creator producer creation date modification date   In addition, the following information is printed: page count encrypted flag (yes/no) print and copy permissions (if encrypted) linearized (yes/no) http://linux.die.net/man/1/pdfinfo    Embed Quote
