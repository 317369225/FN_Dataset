★How can I decide which type of data—mean corrected data or standardized data—is to be used in principal component analysis?
Do both and then see which is more useful for whatever purpose you have in mind. It may make relatively little difference.  E.g. using the example from princomp in R: pcNoScale <-  princomp(USArrests,                        cor = FALSE,                        scale = FALSE,                        scores = TRUE)  pcCor <- princomp(USArrests,                   cor = TRUE,                   scores = TRUE) pcScale <- princomp(USArrests,                     scale = TRUE,                     scores = TRUE) summary(pcNoScale) summary(pcCor) summary(pcScale) loadings(pcNoScale) #Simple structure loadings(pcCor)  #Very messy loadings(pcScale) #Simple structure, identical to NoScale pcNoScale$scores pcCor$scores pcScale$scores and you can compare the scores, loadings or summaries depending on the reason you did PCA,    Embed Quote
