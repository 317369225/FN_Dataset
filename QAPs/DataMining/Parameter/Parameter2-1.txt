★Can function be used as a parameter of another function?
I think this question needs to be clarified. I can see 2 alternatives. 1.- Can you pass the return value of a function as a parameter of a function. Yes. Standard in every programming language I have used. int func() { return 1; } void wow(int i) { } int main(int argc, char *argv[], char *envp[]) { wow(func()); return 0; } 2.- Can you pass a pointer to a function as a parameter of a function. Yes. Standard in C/C++, never used it in other programming languages so I don't know about the rest. int func() { return 1; } void (int (*func)()) { func(); } int main(int argc, char *argv[], char *envp[]) { wow(func); return 0; } EDIT: Thanks to Sergey Zubkov and Vivek Nagarajan for pointing out that you can actually use the function declaration syntax for the function parameter, so this last example can also be written as follows. int func() { return 1; } void (int func()) { func(); } int main(int argc, char *argv[], char *envp[]) { wow(func); return 0; }    Embed Quote
